== Description ==

Capslock plugin will check the caps lock key on or not in the login panel. it will give a message if the caps lock key is on at the time of login.



== Installation ==

1. Upload `capslock.php` to the `/wp-content/plugins/capslock-warning` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
